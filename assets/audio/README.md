Radio audio lives here.

Place only audio you own, created yourself, recorded yourself, or have clear permission to use. Public-domain and royalty-free files are fine when their license allows web use.

The station `src` values in `js/main.js` can point to local files in this folder, for example:

```js
src: "../assets/audio/archive-jazz-leak.mp3"
```

They can also point to direct HTTPS audio stream/file URLs. Do not use YouTube, Spotify, or any service URL that is not a direct browser-playable audio file.

If a station file is missing, the radio page will show a missing-track message. Turn on "synthetic cursed radio mode" on the radio page to generate harmless fake station audio until real tracks are uploaded.
